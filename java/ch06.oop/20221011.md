## ch2-15,16 정수형의 오버플로우

- ex) 계수기(counter) : 인원 수 세는 도구

  - 9999 에서 누르면 0000으로 바뀜 (오버플로우 발생) : 계수기에서 취급하는 자리수가 4자리기때문
    - 2진수가 어려우면 10진수랑 비교하기 : 최대값(1111)+1 > 최소값(0000) , 최소값(0000) -1 > 최대값(1111)

- 10진수

  - 0 ~ 9 : 0 ~ 10-1 개
  - 00 ~ 99 : 0 ~ 100-1 = 0 ~ 10^2-1
  - 000 ~ 999 : 0 ~ 10^3-1
  - 0000 ~ 9999 : 0 ~ 10^4-1

- 2진수

  - 0 ~ 1 : 0 ~ 2-1 개
  - 00 ~ 11 : 0 ~ 2^2-1
  - 000 ~ 111 : 0 ~ 2^3-1
  - 0000 ~ 1111 : 0 ~ 2^4-1
